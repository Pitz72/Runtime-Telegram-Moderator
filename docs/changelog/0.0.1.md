# Changelog - v0.0.1

## [0.0.1] - 2026-03-14

### ✨ Nuove Funzionalità
- **Inizializzazione Monorepo**: Creata la struttura base del progetto con la cartella `backend/`.
- **Backend TypeScript**: Configurato Node.js con TypeScript (Strict Mode) e installate le dipendenze base (`typescript`, `@types/node`, `ts-node`, `nodemon`).
- **Database & ORM**: Implementato **Prisma ORM** con supporto a **SQLite**.
- **Schema Database**: Definiti i modelli relazionali core per il sistema Multi-Bot:
    - `Bot`: Gestione delle istanze bot (Token, stato `isRunning`).
    - `GroupConfig`: Configurazione granulare per bot/gruppo (Captcha, Night Mode, Banned Words).
    - `Log`: Sistema di log centralizzato per azioni di sistema e moderazione.
- **Configurazione Prisma 7**: Adattato il setup alle novità di Prisma 7 utilizzando `prisma.config.ts` per la configurazione del datasource.
